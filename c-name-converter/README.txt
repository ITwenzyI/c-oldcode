# Name Format Converter in C

A small C program that converts a full name from the format `Firstname Lastname` to `Lastname, F.`

## 💡 Description

This project demonstrates:
- Basic character and input handling using `getchar()`
- Simple conditional logic and ASCII checks
- Output formatting in C

The program reads a full name (e.g. "Max Mustermann") from the user, then outputs it in the format "Mustermann, M.".

## 🛠️ Technologies Used

- Language: C 
- Compiler: `gcc`

## 🧾 Example

First and last Name:
Max Mustermann
Mustermann, M.
