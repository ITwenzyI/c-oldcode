# Cashback Calculator in C

A simple C program that calculates the optimal distribution of euro bills (50€, 20€, 10€, 5€) for a given amount of money.

## 💡 Description

This project demonstrates basic logic and pointer usage in C. It calculates how many 50, 20, 10, and 5 euro bills are needed to make up a given amount of cash using as few bills as possible.

The program:
- Takes an integer input (euros)
- Uses a function with pointer parameters to calculate how many bills of each type are needed
- Prints the result in a clear and structured way

## 🛠️ Technologies Used

- Language: C 
- Compiler: `gcc`

## 🧾 Example Output

Enter number of euros: 
135
Chashback:

Euros:      135 Euro
Fifties:    2x
Twenties:   1x
Tens:       1x
Fives:      1x
